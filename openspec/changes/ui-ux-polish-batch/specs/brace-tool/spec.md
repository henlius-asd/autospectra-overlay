## MODIFIED Requirements

### Requirement: 大括号插入工具按钮

系统 SHALL 在工具栏提供"插入大括号"按钮。点击后进入大括号放置模式，用户可在图表上拖拽选择区间放置大括号。

#### Scenario: 点击工具按钮进入放置模式

- **WHEN** 用户点击工具栏中的"插入大括号"按钮
- **THEN** 按钮高亮显示为激活状态，图表光标变为 crosshair，提示用户拖拽选择区间

#### Scenario: 拖拽选择区间

- **WHEN** 在大括号放置模式下，用户在图表上按下鼠标并拖拽
- **THEN** 系统显示拖拽预览（半透明矩形 + 虚线大括号），松开鼠标后弹出标签输入浮层

#### Scenario: 输入标签并确认

- **WHEN** 用户在标签输入浮层中输入文字并点击确认
- **THEN** 大括号被创建并存入 store，放置模式自动退出，按钮恢复非激活状态

#### Scenario: 取消放置

- **WHEN** 在大括号放置模式下，用户按 Escape 键或再次点击工具按钮
- **THEN** 放置模式退出，已记录的端点被清除，按钮恢复非激活状态

#### Scenario: 工具按钮仅在图表有数据时可用

- **WHEN** 图表中没有曲线数据
- **THEN** "插入大括号"按钮显示为禁用状态

## ADDED Requirements

### Requirement: 大括号样式 — I-beam 方括号风格

大括号 SHALL 渲染为 "I-beam" 风格：一条水平线连接两端竖线（类似方括号 [ ]），中间上方有带背景的标签文字。不使用 bezier 花括号。

#### Scenario: 大括号视觉样式

- **WHEN** 一个大括号被创建并显示在图表上
- **THEN** 大括号由以下元素组成：两端竖线（8px 高）、中间水平连接线、上方圆角矩形背景的标签文字

#### Scenario: 大括号颜色

- **WHEN** 大括号显示在图表上
- **THEN** 大括号使用深色线条（如 #555），标签文字与线条同色

### Requirement: 大括号位置动态跟随

大括号的 Y 位置 SHALL 动态计算，位于当前可见曲线中最高曲线的上方 18px 处。不再使用固定的 gridTop 偏移。

#### Scenario: 大括号位于最高曲线上方

- **WHEN** 一个大括号被放置在图表上
- **THEN** 大括书的 Y 位置在可见曲线的最大 Y 值对应像素位置上方 18px

#### Scenario: 大括号位置不超出图表区域

- **WHEN** 最高曲线的 maxY 像素位置靠近图表顶部
- **THEN** 大括号 Y 位置被 clamp 在 gridTop + 5px 以下，不超出图表可视区域

### Requirement: 标签编辑浮层

标签编辑 SHALL 使用 absolute 定位的 HTML 浮层（非 SVG foreignObject），定位在大括号中心上方。

#### Scenario: 浮层定位

- **WHEN** 用户点击大括号编辑标签
- **THEN** 编辑浮层出现在大括号水平中心上方，使用 absolute 定位的 HTML div

#### Scenario: 浮层样式

- **WHEN** 编辑浮层显示
- **THEN** 浮层有白色背景、圆角、阴影、合适的 padding，包含文字输入框和确认/取消/删除按钮
