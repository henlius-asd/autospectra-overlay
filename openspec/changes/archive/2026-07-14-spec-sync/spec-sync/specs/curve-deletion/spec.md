# curve-deletion — Delta Spec

## MODIFIED Requirements

### Requirement: 批量删除曲线

系统 SHALL 支持批量删除已选中的曲线。当过滤后的可见选中曲线（`stagingIds`）数量为 0 时，SHALL 禁用"删除选中"按钮。执行批量删除前 SHALL 弹出 `window.confirm('确定要删除所有选中的曲线吗？')` 确认对话框，用户确认后方可执行删除。

#### Scenario: 批量删除多条曲线

- **WHEN** 用户勾选了多条曲线，点击"删除选中"按钮
- **THEN** 弹出确认对话框"确定要删除所有选中的曲线吗？"，用户确认后所有被勾选的曲线从列表中移除

#### Scenario: 用户取消批量删除

- **WHEN** 用户点击"删除选中"按钮后在确认对话框中点击取消
- **THEN** 不执行删除操作，所有曲线保持不变

#### Scenario: 搜索过滤后无选中曲线时按钮禁用

- **WHEN** 用户在搜索框中输入关键字过滤曲线列表，且当前可见（过滤后）曲线中无选中曲线（`stagingIds` 长度为 0）
- **THEN** "删除选中"按钮处于禁用状态
