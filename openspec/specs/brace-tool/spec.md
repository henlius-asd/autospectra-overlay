# brace-tool Specification

## Purpose
大括号标注工具。通过工具栏按钮触发放置模式，在图表上两次点击创建大括号，支持标签编辑和 dataZoom 联动。

## ADDED Requirements

### Requirement: 大括号插入工具按钮

系统 SHALL 在工具栏提供"插入大括号"按钮。点击后进入大括号放置模式，用户可在图表上通过两次点击（起点和终点）放置大括号。

#### Scenario: 点击工具按钮进入放置模式

- **WHEN** 用户点击工具栏中的"插入大括号"按钮
- **THEN** 按钮高亮显示为激活状态，图表光标变为 crosshair，提示用户点击图表放置第一个端点

#### Scenario: 点击图表放置第一个端点

- **WHEN** 在大括号放置模式下，用户在图表上点击第一个位置
- **THEN** 系统记录起点坐标，显示起点标记，提示用户点击第二个端点

#### Scenario: 点击图表放置第二个端点

- **WHEN** 在大括号放置模式下，用户点击第二个位置
- **THEN** 系统弹出标签输入对话框，起点和终点之间显示大括号预览

#### Scenario: 输入标签并确认

- **WHEN** 用户在标签输入对话框中输入文字并点击确认
- **THEN** 大括号被创建并存入 store，放置模式自动退出，按钮恢复非激活状态

#### Scenario: 取消放置

- **WHEN** 在大括号放置模式下，用户按 Escape 键或再次点击工具按钮
- **THEN** 放置模式退出，已记录的端点被清除，按钮恢复非激活状态

#### Scenario: 工具按钮仅在图表有数据时可用

- **WHEN** 图表中没有曲线数据
- **THEN** "插入大括号"按钮显示为禁用状态

### Requirement: 大括号点击编辑

已创建的大括号 SHALL 支持点击后编辑标签文字或删除。

#### Scenario: 点击大括号编辑标签

- **WHEN** 用户点击已有大括号的标签文字
- **THEN** 弹出编辑对话框，显示当前标签文字，用户可修改或删除

#### Scenario: 删除大括号

- **WHEN** 用户在编辑对话框中点击删除
- **THEN** 该大括号从 store 中移除，图表中不再显示

### Requirement: 大括号随 dataZoom 联动

大括号 SHALL 在 dataZoom 缩放/平移时保持与曲线的相对位置不变。大括号的像素坐标 SHALL 通过 ECharts `convertToPixel` API 实时计算。

#### Scenario: 缩放时大括号位置同步

- **WHEN** 用户通过 dataZoom 缩放图表
- **THEN** 大括号的像素位置随缩放比例同步更新，与曲线无可见脱节（视觉误差 < 2px）
