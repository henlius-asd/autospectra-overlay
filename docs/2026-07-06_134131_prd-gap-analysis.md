### [TODO-9] 大括号创建 Bug 修复

- **原始分析**：大括号标注无法创建
- **根因分析**：
  - [BraceOverlay.tsx:122](src/components/chart/BraceOverlay.tsx#L122) `className="... pointer-events-none"` → Tailwind 类设置 `pointer-events: none`
  - [BraceOverlay.tsx:126](src/components/chart/BraceOverlay.tsx#L126) `style={{ pointerEvents: isDrawing ? 'auto' : 'none' }}` → 初始 `isDrawing=false` 时也是 `none`
  - **双重 `pointer-events: none` 导致 SVG 完全透明于鼠标事件** → `onMouseDown` 永远无法触发 → `isDrawing` 永远无法变为 `true` → 死锁
- **修复方向**：
  - 移除 `pointer-events-none` CSS 类
  - SVG 始终 `pointer-events: auto`，在事件处理函数中根据 Shift 键状态决定是否响应
  - 或者：改为专门的"插入大括号"工具按钮触发创建模式
- **确认结论**：建议加入一个工具按钮专门来插入大括号，插入后可以输入标签
- **涉及文件**：[BraceOverlay.tsx](src/components/chart/BraceOverlay.tsx)、可能需要新增工具栏按钮
- **状态**：✅ 已修复（2026-07-06）
